# LinkGuard
LinkGuard is a user-friendly web app that allows anyone to instantly check the safety of a URL.
It evaluates links using multiple heuristic checks including:
   >HTTPS usage
   >Domain age (new domains flagged as risky)
   >URL length (long URLs may indicate phishing)
   >Suspicious keywords (e.g., login, verify, free)
   >IP-based URLs

The app provides a risk score, a human-readable explanation, and visual feedback via an animated risk meter.
